=================
Plasma Calculator
=================

.. currentmodule:: plasmapy.utils.calculator


Overview
--------

Plasma calculator is an interactive notebook installed alongside PlasmaPy that
allows users to input a set of plasma properties and get immediate output for
various functions at once.

Using Plasma Calculator
-----------------------

To invoke the app use ``plasma_calculator``, by default opens in a browser bootstrapping
the notebook in a light theme and uses port 8866 by default.

``plasma-calculator`` takes optional arguments such as ``--dark``, ``--port``, ``--no-browser`` pass flag
``-h`` or ``--help`` to get full list of supported arguments.
